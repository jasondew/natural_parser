 —  Release Notes for Natural Version 6.3 for UNIX  —

Supported Operating Systems

The operating systems listed below are supported with this Natural version.

  • Solaris 10 - 64 bit (Sun and FSC)

  • AIX 5.3 - 64 bit

  • HP-UX 11.i v2 64 bit (PA-RISC)

  • HP-UX 11.i v2 64 bit (Itanium)

  • SUSE Linux Enterprise Server 10 (IA32)

  • SUSE Linux Enterprise Server 10 (z/Linux)

  • SUSE Linux Enterprise Server 10 for AMD64 and Intel EM64T (x86-64)

  • Red Hat Enterprise Linux 4.0 (IA32)

  • Red Hat Enterprise Linux 4.0 (z/Linux)

  • Red Hat Enterprise Linux 4.0 for x86-64 (AMD64, EM64T)

See also: Product Requirements in the Installation documentation.
