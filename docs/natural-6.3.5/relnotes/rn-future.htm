Notice of Future Changes

The following changes are planned for future versions of Natural.

Remote Debugging

Remote debugging (that is debugging a native Natural for UNIX application from a Windows computer) will no longer be supported. The Windows client will no longer be delivered.

Debugging a UNIX application will only be possible with Natural Studio when you are working in a remote UNIX environment using SPoD.
