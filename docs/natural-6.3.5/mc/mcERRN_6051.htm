 —  Messages and Codes  —

Natural System Error Messages 6051-6100

NAT6098: This statement is not implemented in the current version.

Text This statement is not implemented in the current version.
Expl. This statement is not available in the current version of Natural.
Actn. Remove this statement from the program.

NAT6099: File-open error :1: occurred for :2:.

Text File-open error ... occurred for ....
Expl. DU=ON causes dump-file generation into the directory defined
in the Natural TMP directory option in the Natural Configuration
utility.
The "fopen" call failed with the error indicated.
The dump file is either already open or the user has no write
permission.
Actn. See the operating-system manual for further information.

NAT6100: Failed to open XREF dump file.

Text Failed to open XREF dump file.
Expl. The XREF dump file is either already open or the user has no write
permission.
Actn. Close XREF dump file or set write permission.
