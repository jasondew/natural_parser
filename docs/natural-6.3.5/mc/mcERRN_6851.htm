Natural System Error Messages 6851-6900

NAT6890: The map source cannot be loaded into the editor.

Text The map source cannot be loaded into the editor.
Expl. This map cannot be read by the map editor, because the map source
contains an invalid Natural syntax or the internal definitions used
by the map editor cannot be resolved.
Actn. In most cases that is an internal error.
Contact Software AG Support.
