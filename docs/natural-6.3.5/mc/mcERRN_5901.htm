Natural System Error Messages 5901-5950

NAT5901: Cannot allocate internal reader.

Text ESY5901 Cannot allocate internal reader.
Expl. Dynamic allocation of internal reader failed.
Actn. Contact the systems programmer.

NAT5902: Dynamic allocation failed.

Text ESY5902 Dynamic allocation failed.
Actn. Contact the systems programmer.

NAT5903: Dynamic allocation failed for temporary data set.

Text ESY5903 Dynamic allocation failed for temporary data set.
Expl. Dynamic allocation for a temporary dataset failed.
Actn. Contact the systems programmer.

NAT5904: Edit not allowed (SCLM protection).

Text ESY5904 Edit not allowed (SCLM protection).
Expl. The selected PDS member has been protected by SCLM.
Only browsing if this member is allowed, all update
actions are denied.
Actn. Use SCLM dialog to free member.

NAT5906: DESERV :1: error, R15=:2:, R0=:3:.

Text ESY5906 DESERV :1: error, R15=:2:, R0=:3:.
Expl. A DESERV macro was issued and an unexpected error occurred. The
function and values for R15 and R0 are shown.
Actn. Check the R15 and R0 values and if necessary contact Software AG
technical support.

NAT5907: DSPSERV :1: error, R15=:2:, R0=:3:.

function and values for R15 and R0 are shown.
Actn. Check the R15 and R0 values and if necessary contact Software AG
technical support.

NAT5908: ALESERV :1: failure, R15=:2:, R0=:3:.

Text ALESERV :1: error, R15=:2:, R0=:3:.
Expl. An ALESERV macro was issued and an unexpected error occurred. The
function and values for R15 and R0 are shown.
Actn. Check the R15 and R0 values and if necessary contact Software AG

NAT5909: Syntax error in JCL command.

Text ESY5909 Syntax error in JCL command.
Expl. A syntax error was detected.
Actn. Check JCL and correct error.
