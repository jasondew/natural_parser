OPTIONS

OPTIONS parameter

This document covers the following topics:

  • Function

Function

The OPTIONS statement can be used to specify compilation options as parameters for the current Natural programming object. These are the same options that can be specified within a Natural session with the COMPOPT system command.

Note:
No mainframe-specific options are available. For compatibility reasons, for example when programming a cross-platform application, such options are ignored during compile time.
